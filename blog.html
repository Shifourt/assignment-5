1. What is DOM? What is the purpose of DOM?

If we talk about DOM, first we should know the full meaning of DOM. Document Object Model(DOM) is the interface for all the web documents. It helps us change the structure, style and content of the web document. it shows the documents as nodes and objects so that programming languages can interact with the page.

2. How to select element from DOM?

there are few away we can select elements from DOM. they are given below-

  • getElementsByTagName()
  • getElementsByClassName()
  • getElementById()
  • querySelector()
  • querySelectorAll()

3. What is event Bubble?

Event Bubble is a concept. it happens when an element receives an event and that event goes up to the root of the element or root of the DOM Tree.

4. What is callback function? What does it do?

Callback function is a function that is passed as an argument to another function. Callback function makes a function run when another function has already completed its execution.
